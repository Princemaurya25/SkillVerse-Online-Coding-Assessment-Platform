# 💻 SkillVerse – Online Coding & Assessment Platform

SkillVerse is an online coding practice and assessment platform where students can improve programming skills through coding challenges and quizzes.

## 🚀 Features

### Students

- Registration & Login
- Coding Challenges
- Online MCQ Tests
- Leaderboard
- Certificates
- Progress Tracking

### Instructor

- Manage Courses
- Add Coding Problems
- Create Quizzes
- View Student Progress

### Admin

- Dashboard
- User Management
- Contest Management
- Reports
- Analytics
